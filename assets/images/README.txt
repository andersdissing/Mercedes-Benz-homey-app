Image Assets Required
=====================

Please add the following images to this directory:

1. small.png - 250x175 pixels (app store listing thumbnail)
2. large.png - 500x350 pixels (app store listing image)
3. xlarge.png - 1000x700 pixels (app store listing large image)

These should be PNG images with the Mercedes-Benz logo or a car icon.

For development, you can use placeholder images, but they must be the exact dimensions specified above.

Recommended content:
- Mercedes-Benz logo/icon
- Or a generic car silhouette
- Use official Mercedes-Benz brand colors (silver, black)

Note: Make sure you have rights to use any images/logos before publishing to the Homey App Store.
